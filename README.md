
# Netflix Content Analysis Dashboard

## Project Overview

This project analyzes Netflix's content library using Excel, SQL, and Power BI. The objective was to clean raw data, perform exploratory data analysis, and build an interactive dashboard to uncover trends in Netflix movies and TV shows.

## Tools Used

* Microsoft Excel (Data Cleaning & Preprocessing)
* SQL (Data Analysis & Queries)
* Power BI (Dashboard Development)

## Dataset

* Total Records: 8,809+
* Content Types: Movies & TV Shows
* Features: Title, Director, Cast, Country, Date Added, Release Year, Rating, Duration, Genre

## Data Cleaning

* Removed duplicates
* Handled missing values
* Standardized date formats
* Cleaned country information
* Corrected data inconsistencies
* Prepared data for SQL and Power BI analysis

## SQL Analysis

Key analyses performed:

* Movies vs TV Shows count
* Top countries by content
* Content added by year
* Rating distribution
* Genre analysis
* Release year trends

## Dashboard Features

### KPI Cards

* Total Titles
* Total Movies
* Total TV Shows
* Total Countries

### Interactive Visualizations

* Movies vs TV Shows Distribution
* Content Added by Year
* Top Countries by Content
* Ratings Distribution
* Movies Released by Year
* Country-wise Content Analysis

### Filters

* Type
* Country
* Release Year
* Rating
* Genre

## Key Insights

* Movies dominate Netflix's content library.
* The United States contributes the highest number of titles.
* Content additions increased significantly after 2015.
* TV-MA and TV-14 are among the most common ratings.
* Netflix content growth peaked around 2019–2020.
